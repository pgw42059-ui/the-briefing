
# 해외선물 투자자용 인사이트 대시보드

## 개요
나스닥, 항셍, 골드, 오일 등 해외선물을 거래하는 한국인 투자자를 위한 경제지표 분석 대시보드입니다. 로그인 없이 바로 사용할 수 있으며, 주요 경제 일정과 지표를 한눈에 확인하고 강세/약세를 판단할 수 있습니다.

## 핵심 기능

### 1. 메인 대시보드
- 주요 해외선물 종목(나스닥, 항셍, 골드, 오일) 현재 시세 카드 표시
- 전일 대비 등락률, 변동폭 등 핵심 정보를 한눈에 확인
- 다크 테마 기반의 트레이딩 친화적 UI

### 2. 경제 캘린더
- 주요 경제지표 발표 일정을 캘린더/리스트 형태로 표시
- 지표 중요도 표시 (⭐ 높음 / 보통 / 낮음)
- 발표 예정 시간, 예상치, 이전치 정보 제공
- 한국 시간(KST) 기준으로 표시

### 3. 종목별 강세/약세 분석 화면
- 각 종목(나스닥, 항셍, 골드, 오일)별 상세 페이지
- 주요 경제지표가 해당 종목에 미치는 영향 요약
- 강세/약세 시그널을 시각적으로 표시 (게이지, 아이콘 등)
- 간단한 차트로 최근 추세 시각화

### 4. 시황 요약 섹션
- 오늘의 주요 이슈/이벤트 요약
- 종목별 주의사항 및 핵심 포인트 표시

## 디자인 방향
- 다크 모드 기반의 깔끔한 트레이딩 UI
- 모바일 반응형 (스마트폰으로도 편하게 확인)
- 한국어 인터페이스
- 빨간색(상승)/파란색(하락) 한국 증권 컬러 컨벤션 적용

## 기술 구현
- 외부 금융 데이터 API를 통해 실시간 시세 및 경제지표 데이터 연동
- Supabase Edge Function을 활용한 API 프록시
- 자동 새로고침으로 최신 데이터 유지
